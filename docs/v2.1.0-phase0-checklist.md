## 附录：验证命令速查

### 功能验证
```bash
# 1. Tauri 启动
cd "D:\Pangu Nebula"
cargo tauri dev

# 2. Sidecar PoC
pytest tests/test_sidecar_poc.py -v

# 3. /health/ready
curl -H "Authorization: Bearer <TOKEN>" http://127.0.0.1:<PORT>/health/ready

# 4. Bearer token 认证（无 token 应返回 401）
curl http://127.0.0.1:<PORT>/health/ready

# 5. 优雅关闭
curl -X POST -H "Authorization: Bearer <TOKEN>" http://127.0.0.1:<PORT>/shutdown
```

### 回归测试
```bash
# 全量 pytest
pytest tests/ -v

# Rust 模块测试
pytest tests/test_rust_modules.py -v

# 前端 E2E
cd frontend && npm run e2e

# Rust 双目标编译
cd rust/browser_use && cargo build --lib && cargo build --lib --features python
cd rust/computer_use && cargo build --lib && cargo build --lib --features python
```

### 安全验证
```bash
# cargo audit
cd src-tauri && cargo audit --deny warnings

# Sidecar 完整性
python scripts/gen_sidecar_hash.py  # 生成 sidecar.sha256
# 篡改 sidecar.exe → cargo tauri dev 应拒绝启动
```

### 打包验证

# PyInstaller sidecar
python scripts/build_sidecar.py

# Tauri 打包
cargo tauri build

# 签名（Windows）
signtool sign /a /fd sha256 /tr http://timestamp.digicert.com /td sha256 <file>.msi
